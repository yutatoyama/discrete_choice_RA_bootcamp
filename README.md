## 必要なパッケージ

以下のパッケージについてのインストールをお願いします。

```
library("tidyverse")
library("knitr")
library("mlogit") # ロジットモデル推定のためのパッケージ
library("stargazer") # 推定結果の表作成
library("optimx") # 数値最適化のためのパッケージ
```
